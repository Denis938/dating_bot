Data Flow Examples
==================

1. USER REGISTRATION (Telegram)
───────────────────────────────
Telegram → Bot → /auth/register → Verify initData → Create User → JWT → Profile


2. MATCHING FLOW
────────────────
User A → GET /matching/recommendations → Filter by preferences → Return profiles
User A → POST /matching/like (User B) → Check mutual like → Create Match → Notify


3. CHAT MESSAGE FLOW
────────────────────
User A → WebSocket → Send Message → Save to DB → Redis Pub/Sub → User B receives


4. PHOTO UPLOAD FLOW
────────────────────
User → POST /media/upload → Validate → Generate presigned URL → Upload to S3 → Save to DB
